Was ist Spring?

  • 🍃 Open-Source Java-Framework (seit 2002)
  • 🏗️ Hilft beim Bauen von großen Anwendungen
  • 🔧 Macht viele Dinge automatisch – du schreibst weniger Code
  • 🌍 Weltweit das #1 Framework für Java-Backends
  • 🚀 Basis für Spring Boot, MVC, Security & mehr
💡

Metapher

Spring ist wie ein erfahrener Assistent: Du sagst, was du brauchst – Spring sorgt dafür, dass alles bereitsteht.

Inversion of Control IoC

Wer hat die Kontrolle?

❌ Ohne Spring
Car car = new Car();
Engine e = new Engine();
car.setEngine(e);

// Du steuerst alles

Du erzeugst jedes Objekt von Hand

→
✅ Mit Spring (IoC)
@Component
class Car { ... }

@Component
class Engine { ... }

// Spring steuert alles

Spring übernimmt die Kontrolle

Dependency Injection DI

Spring liefert dir, was du brauchst – automatisch!

CarService.java
@Service
public class CarService {

  // Spring injiziert automatisch ↓
  @Autowired
  private EmailService emailService;

  public void buyCar(Car car) {
    // einfach benutzen!
    emailService.sendConfirmation(car);
  }
}
@Service

Markiert eine Klasse als Spring-Dienst

@Autowired

Spring sucht & injiziert die Abhängigkeit

Kein new!

Du erzeugst keine Objekte per Hand mehr

REST Controller

Deine App spricht mit der Welt – via HTTP

CarController.java
@RestController
@RequestMapping("/api/cars")
public class CarController {

  @Autowired
  private CarService carService;

  // GET /api/cars → alle Autos
  @GetMapping
  public List<Car> getAll() {
    return carService.findAll();
  }

  // POST /api/cars → neues Auto
  @PostMapping
  public Car add(@RequestBody Car car) {
    return carService.save(car);
  }
}
GET Daten lesen
POST Daten erstellen
PUT Daten ändern
DELETE Daten löschen
